# ⚡ SemaBuzz.me

> **Built for the desktop. Made for the wire.** > A minimalist, P2P live-typing utility that visualizes the "Buzz" of human connection.

---

## 📻 The Concept
**SemaBuzz** is a C#/.NET 9 desktop application that turns digital messaging into a tactile experience. Using the custom **SemaBuzz Protocol**, characters are streamed character-by-character across a direct P2P connection. 

No servers. No cloud. Just the raw, rhythmic pulse of a live wire between you and a peer.

## ✨ Signature Features
* **The SemaBuzz Protocol:** A high-frequency, stateful P2P stream designed for zero-latency synchronization.
* **The Buzz Indicator:** A flickering, neon "filament" UI that vibrates in real-time as data packets arrive.
* **Live-Wire Typing:** See the thought process. Text appears as it is typed, creating a raw and intimate "echo" of the sender.
* **Obsidian & Amber Aesthetic:** A high-contrast, Lofi-Industrial WPF interface designed for the focused desktop user.

## 🛠️ Tech Stack
* **Language:** C# 12 / .NET 9
* **Framework:** WPF (Windows Presentation Foundation)
* **Networking:** P2P via **SemaBuzz Protocol** (TCP/UDP Hole Punching)
* **Security:** AES-256 end-to-end encryption.
